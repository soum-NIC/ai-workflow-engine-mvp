# Components Reference

Atomic UI instances engineered with strict token limits. Hosted in `/apps/frontend/components/ui/`.

## General Use Forms & Inputs
- **Button.tsx**: Variants: `primary`, `secondary`, `ghost`. Used everywhere. "Next Best Action" strictly maps to Primary.
- **Input.tsx**: Transparent bordering with Ring highlighting on focus. Inline error variants built-in.
- **Card.tsx**: Standard bounding block for all application workflows. Features built in spatial `.gap-4`.

## Interactive Surfaces
- **Modal.tsx**: Dark overlay, center pivot point, pure H1/H2 header hierarchies.
- **Toast.tsx**: Bottom-right flyouts tracking structural engine events (e.g. `Workflow Autonomously Finished`).
- **SidebarNavigation.tsx**: Left bounds standard layout wrapper. Ghost-variant interaction elements mapping to Dashboard links.
